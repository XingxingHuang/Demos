### html 

It's the structure of websites. It defines the content of web pages.

every web page on the Internet is secretly written in HyperText Markup Language, otherwise known as "HTML". Your browser reads HTML for you and just shows you the pretty version.

### css
It's the appearance of webistes. It specifies the layout of web pages.

The Cascading Style Sheet (CSS) describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.
 

### javascript
It's the functions of websites. It program the behavior of web pages.

Javascript often abbreviated as JS, is a high-level, dynamic, weakly typed, object-based, multi-paradigm, and interpreted programming language.

`JavaScript` and `Java` are completely different languages, both in concept and design.
